# HERS_Project

## Checking out project from Github
```
# Choose Directory for project
cd <folder-you-want-your-project-in>

# Checkout your project
git checkout https://github.com/pmanko/HERS_Project.git
```

## Workflow
```
# Pull in the latest changes/work of other people 
git pull

# Do lots of awesome work

# Tell git you want to keep track of the work you just did
git add .

# Commit your work and describe what you did
git commit -m "describe all of the awesome work you did"

# Push your work to the Github repo so others can have it
git push
```
